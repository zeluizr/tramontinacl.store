# Store Components

Este diretório contém os componentes personalizados utilizados no storefront da loja **Armazém do Sopro**.

## Componentes disponíveis

- [PixDiscounts](./PixDiscounts.md): Exibe o valor com desconto aplicado para pagamentos via Pix.

## Como usar

Cada componente é documentado separadamente no diretório `docs`. Acesse os arquivos individuais para instruções de uso, propriedades disponíveis e exemplos.
